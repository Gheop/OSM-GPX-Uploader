# Contributing

## Getting Started
1. Fork the repository
2. Clone: `git clone https://github.com/YOUR_USERNAME/OSM-GPX-Uploader.git`
3. Install: `pip install -r requirements-dev.txt`

## Running Tests
```bash
make test
make test-cov
```

## Code Style
```bash
make format
make lint
```

## Pull Requests
1. Create a branch
2. Make changes
3. Run tests
4. Submit PR

Thank you! ❤️
